# Soporte

Para uso y troubleshooting, revisa primero el [README](README.md), la
[guía del usuario](docs/USER_GUIDE.md) y [recuperación](docs/RECOVERY.md).

Abre un issue si puedes aportar versión, sistema operativo, comando exacto, salida redactada y fixture
mínima. No compartas secretos. Preguntas de producto, selección de stack y operación de servicios externos
pertenecen al proyecto consumidor.

El proyecto es mantenido con presupuesto bajo y no ofrece soporte comercial ni tiempos de respuesta
garantizados.
